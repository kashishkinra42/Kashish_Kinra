Hand Gesture Recognition

Developed a volume control system using OpenCV and Python that leverages hand gestures via webcam to adjust system volume. The system recognizes specific gestures, allowing users to control volume through non-verbal interactions. This project demonstrates the potential of gesture-based control for intuitive and hands-free computer interaction. The findings were published in a journal on Human-Computer Interaction, showcasing advancements in gesture recognition technology and its application in everyday computing tasks. The system highlights innovative approaches to user interface design and contributes to the field of Human-Computer Interaction by exploring new methods of control and interaction.

Restaurant Reviewing using NLP

This machine learning model forecasts whether a given customer review will be favorable or unfavorable for a restaurant. It combines the techniques of count vectorizer and natural language processing (NLP) to analyze the sentiment of reviews. The count vectorizer transforms textual data into numerical features, while NLP techniques process and interpret the meaning behind the words. By training on a dataset of restaurant reviews, the model can predict the sentiment of new reviews, providing valuable insights for restaurant management to understand customer feedback and improve service. This approach enhances decision-making in the hospitality industry by leveraging AI-driven sentiment analysis.
